FILE* open_json(char* dossier, char* name, char* mode)

Créer, s'il ne l'est pas, le fichier JSON et l'ouvre.

Parameters
dossier Chaine de caractère représentant le dossier ou se trouve le json.
name Chaine de caractère représenant le nom du fichier sans le '.json'.
mode Mode d'ouverture du fichier.
Returns Un pointeur sur le fichier ouvert.

t_erreur del_json(char* dossier, char* name)

Supprime un fichier JSON.

Parameters
dossier Chaine de caractère représentant le dossier ou se trouve le json.
name Chaine de caractère représenant le nom du fichier sans le '.json' à supprimer.
Returns Une erreur s'il y en a une.

t_erreur write_json(FILE* file, char* key, void* value, char* value_type)

Ecrit une clé/valeur au format JSON dans un fichier.

Parameters
file Le fichier ou écrire.
key La clé correspondant à la valeur.
value La valeur à écrire.
value_type Le type de la valeur à écrire.
Returns Une erreur s'il y en a une.

L'écriture s'effectue dans le fichier 'file' en ajout avec pour contenu le couple clé/valeur 'key:value'.

t_erreur open_json_obj(FILE* file)

Démarre un nouvelle objet au format JSON dans un fichier.

Parameters
file Le fichier ou écrire.
Returns Une erreur s'il y en a une.

L'écriture s'effectue dans le fichier 'file' en ajout avec pour contenu '{' et la variable crt_car est mis à jour.

t_erreur close_json_obj(FILE* file)

Ferme l'objet JSON.

Parameters
file Le fichier ou écrire.
Returns Une erreur s'il y en a une.

L'écriture s'effectue dans le fichier 'file' en ajout avec pour contenu '}' et la variable crt_car est mis à jour.

t_erreur extract_json_obj(FILE* file, char** obj)

Récupère un objet JSON dans un fichier.

Parameters
file Le fichier à lire.
obj L'endroit ou enregistrer l'objet.
Returns Une erreur s'il y en a une.

La lecture s'effectue dans le fichier 'file' et on récupère un objet.
